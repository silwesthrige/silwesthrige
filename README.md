### 🧠 Engineering Mindset

> *"Code is a liability. Clean, tested, and maintainable code is an asset."*

I'm a Computer Science candidate at **NIBM Kurunegala** with a product-focused approach to software engineering. I specialize in **Flutter** for cross-platform dominance and **Next.js/Node.js** for full-stack resilience. I don't just write features; I architect solutions that scale gracefully and degrade safely.

- 🏗️ **Currently Building:** High-availability systems with **Next.js** and edge-ready **Node.js** microservices.
- 📐 **Architecture Focus:** Clean Architecture, Repository Pattern, and State Management (BLoC/Riverpod).
- 🤖 **R&D Interest:** Integrating lightweight **LLM agents** into local-first mobile experiences.
